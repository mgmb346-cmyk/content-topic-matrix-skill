# 内容选题矩阵 Skill

一个适合内容创作者使用的 Codex Skill：输入一个母题，按「内容纵深 × 呈现形式」拆成选题池，再用五问筛选出本周最值得拍的内容。

它不只适合 AI 博主，也适合女性成长、学习、生活、职场、个人 IP、小红书、抖音、B站、视频号等内容创作者。

## 它解决什么问题

很多人不是没有灵感，而是没有稳定拆选题的结构。

这个 Skill 会把一个母题拆成：

- 20 个选题抽屉
- 每条选题的标题、开头钩子、核心看点
- 五问筛选评分
- 本周最值得拍的 3 条建议

## 核心矩阵

内容纵深：

- 新手入门
- 进阶技巧
- 趋势判断
- 避坑指南
- 决策共鸣

呈现形式：

- 干货教程
- 故事案例
- 观点解读
- 互动话题

## 五问筛选

- 新鲜度：这个选题有没有最近才值得讲的理由？
- 可实测：我能不能真的操作一遍，并拿出结果？
- 可讲清：这条能不能在 60-120 秒内讲明白？
- 可互动：用户看完有没有话可以回？
- 可延展：如果数据好，我能不能继续拍 3-10 条？

## 安装方式

把 `content-topic-matrix` 文件夹复制到你的 Codex skills 目录：

```bash
cp -R content-topic-matrix ~/.codex/skills/
```

重启 Codex 后即可使用。

## 使用方式

你可以这样说：

```text
用内容选题矩阵 Skill，帮我拆这个母题：

账号赛道：女性成长
账号风格：清醒、温柔但有力量
目标人群：被原生家庭消耗、想独立但还没准备好的女生
平台：小红书 / 抖音
母题：女性如何逃离原生家庭
本周目标：收藏、评论、涨粉
已有素材：个人故事 / 观察 / 方法论
```

也可以更简单：

```text
用内容选题矩阵 Skill，帮我拆母题：下班后2小时
```

## 说明

这个 Skill 的重点不是让 AI 随便想标题，而是让 AI 执行一套固定的选题系统：

> 母题 → 选题矩阵 → 五问筛选 → 本周可拍清单
