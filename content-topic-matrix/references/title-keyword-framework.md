# Title Keyword Framework

Use this reference when optimizing titles for any niche.

Do not hardcode one niche's high-frequency words into every title. Build or infer a niche-specific keyword bank first, then use it selectively.

## Core Principle

Title optimization is not "add more hot words."

It is:

> Match the user's search/click language while keeping the account identity clear.

Use high-frequency words as reference signals, not mandatory ingredients.

## Four Keyword Buckets

For any niche, organize keywords into four buckets.

### 1. Core search terms

Words users actively search or immediately recognize.

Examples:

- AI niche: 免费 AI, 国产 AI, AI 脚本, 剪映 AI, DeepSeek, 可灵, AI 提示词
- Female growth: 情绪稳定, 内耗, 高敏感, 原生家庭, 亲密关系, 边界感
- Study: 学习方法, 复盘, 背书, 拖延, 专注力, 记忆力, 学习计划
- Lifestyle: 独处, 一人食, 租房改造, 周末生活, 早起, 收纳, 低成本变精致
- Workplace: 汇报, 沟通, 领导, 同事关系, 办公提效, 升职, 职场新人

### 2. Emotional click hooks

Words that create stopping power or emotional recognition.

Examples:

- General: 救命, 绝了, 藏私, 没人讲, 亲测, 实测, 普通人, 小白, 千万别踩坑
- Female growth: 清醒, 扎心, 终于懂了, 别再忍, 慢慢变好, 自救
- Study: 开窍, 后悔才知道, 真的有用, 别再假努力, 普通人也能做到
- Lifestyle: 松弛感, 治愈, 低成本, 不费钱, 普通女生, 变舒服
- Workplace: 终于会了, 别再硬扛, 职场新人必看, 老板爱听, 少走弯路

### 3. Number and time cues

Words that make a title concrete.

Examples:

- 3 分钟
- 5 个
- 7 天
- 10 秒
- 一键
- 一天
- 全套
- 第 3 个
- 90%
- 2026 最新

Only use a number or time cue when it can be supported by the content.

### 4. Pain/result keywords

Words that connect the title to user demand.

Examples:

- AI niche: 起号涨粉, 不用出镜, 省时间, 批量出片, 办公提效, 零基础做内容
- Female growth: 停止内耗, 情绪稳定, 建立边界, 走出关系消耗, 找回自己
- Study: 坚持学习, 提高效率, 不再拖延, 记得更牢, 复习不崩
- Lifestyle: 恢复能量, 下班后不废掉, 房间变舒服, 低成本生活, 生活有秩序
- Workplace: 汇报清楚, 沟通不内耗, 提高效率, 不被甩锅, 获得认可

## Rewrite Formula

Use:

> Core search term + user identity/pain + concrete result + proof/time/number

Not every title needs all four pieces.

## Rewrite Rules

- Keep one clear primary keyword.
- Add one user identity or pain point.
- Add one concrete result.
- Add proof words only when true: 实测, 亲测, 7 天, 我试了.
- Use at most 2-3 hook words per title.
- Avoid fake certainty, exaggerated income claims, and unsupported numbers.
- If a hot word attracts the wrong audience, remove it.

## Examples

AI:

- Weak: 《AI 内容创作方法论》
- Better: 《不知道拍什么？我用 AI 3 分钟拆出 20 个选题》

Female growth:

- Weak: 《关于情绪稳定的思考》
- Better: 《高敏感女生停止内耗，先别急着变冷漠》

Study:

- Weak: 《如何提高学习效率》
- Better: 《别再假努力了，我用这张复盘表坚持了 7 天》

Lifestyle:

- Weak: 《我的周末生活》
- Better: 《普通女生低成本过周末，真的能把状态养回来》

Workplace:

- Weak: 《职场沟通技巧》
- Better: 《职场新人汇报别再硬讲过程，领导更想听这 3 点》
